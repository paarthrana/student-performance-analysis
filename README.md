# 📊 Student Performance Analysis

A beginner-friendly Data Science project built using **Python**, **Pandas**, **NumPy**, and **Matplotlib** to analyze and visualize student performance data.

---

## 🚀 Features

- Load student data from a CSV file
- Display dataset information
- Calculate:
  - Total number of students
  - Average marks
  - Highest marks
  - Lowest marks
  - Average attendance
  - Highest attendance
  - Lowest attendance
- Identify the top scorer(s)
- Display students scoring above 90 marks
- Generate summary statistics using Pandas
- Visualize data using charts *(coming in the next update)*

---

## ▶️ How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install the dependencies

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python analysis.py
```
